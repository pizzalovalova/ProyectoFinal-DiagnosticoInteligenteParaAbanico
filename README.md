# ProyectoFinal-DiagnosticoInteligenteParaAbanico
Un diagnóstico inteligente para abanico basado en ESP32 combina IoT y domótica para monitorear el estado del motor, la temperatura ambiente y medir el consumo eléctrico. Utiliza sensores para automatizar el encendido, detectar fallas y enviar alertas al celular o plataformas domóticas.
